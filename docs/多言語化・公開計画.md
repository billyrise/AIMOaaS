# AIMOaaS 公式サイト 多言語化・公開計画

## 1. 前提と目標

| 項目 | 内容 |
|------|------|
| **サイト** | AIMOaaS 公式サイト（aimoaas.com） |
| **ホスティング** | Cloudflare Pages |
| **現状** | 単一 `index.html`（日本語のみ・約720行） |
| **目標** | 多言語化し、英語・日本語を運用。LP（lp.aimoaas.com）の言語構成に揃える。 |

---

## 2. 言語とURL設計（LPに揃える）

LP（https://lp.aimoaas.com/）は次のようなパスで運用されていると想定します。

| 言語 | LPのURL | 本サイト（aimoaas.com）のURL |
|------|---------|------------------------------|
| **英語** | `https://lp.aimoaas.com/`（ルート） | `https://aimoaas.com/`（ルート・サフィックスなし） |
| **日本語** | `https://lp.aimoaas.com/ja/` | `https://aimoaas.com/ja/` |

- **英語**: ルートのみでシンプルなサフィックス（パスなし）。
- **日本語**: 言語が分かるサフィックス `/ja/` を使用。
- 今後LPに他言語（例: `/zh/` など）を追加する場合は、同じルールで本サイトにも追加する。

---

## 3. フォルダ・ファイル構成（実装済み）

```
AIMOaaS/
├── index.html              # 英語版（ルート = デフォルト）
├── aimo-standard/
│   └── index.html          # 英語版 AIMO Standard ページ
├── ja/
│   ├── index.html          # 日本語版トップ
│   └── aimo-standard/
│       └── index.html      # 日本語版 AIMO Standard ページ
├── assets/
│   └── aimo-standard-unifying-regulations.png
├── _redirects              # 任意
└── docs/
    └── 多言語化・公開計画.md
```

- **ルート `index.html`**: 英語コンテンツ。`<html lang="en">`、英語用 title/description。
- **`ja/index.html`**: 現在の日本語コンテンツ。`<html lang="ja">`、日本語用 title/description のまま。
- Cloudflare Pages は静的配信のため、`/` → `index.html`、`/ja/` → `ja/index.html` が自動で対応。

## 6. 注意事項・確認事項

1. **LPの言語一覧**  
   現時点では LP の「/」と「/ja/」を前提にしています。LP に他言語がある場合は、そのパス（例: `/zh/`）を同じ規則で本サイトに追加してください。

2. **既存リンク・共有URL**  
   これまで `https://aimoaas.com/` を日本語として共有している場合、公開後はルートが英語になるため、日本語は `https://aimoaas.com/ja/` で案内する必要があります。必要に応じてルートから `/ja/` へのリダイレクトも検討してください。

3. **メール・問い合わせ**  
   `mailto:contact@aimoaas.com` の subject/body は、英語版では英語文に変更することを推奨します。

4. **404**  
   存在しないパスで 404 を返したい場合は、Cloudflare Pages の出力に `404.html` を置くと、SPA ではなく静的サイトとして 404 が返ります。
